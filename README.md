MinerDash

Config
--------------

Open index.php file and change following things.
```php
// API Key from ypool.net
$api_key = 'XXXXXXXXXXXXXXXXXXXXXXXX';
// Coins: XPM, FTC, PTS, DOGE, MTC, RIC
$coins = array('XPM', 'RIC', 'DOGE');
```
